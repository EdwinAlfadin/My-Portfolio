## 🚀 Cara Menjalankan

Clone repository

bash
git clone https://github.com/EdwinAlfadin/My-Portfolio/my-portfolio.git


Masuk ke folder

bash
cd my-portfolio


Install dependency

bash
npm install


Jalankan project

bash
npm start


Buka browser


http://localhost:3000
